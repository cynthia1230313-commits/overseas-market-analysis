# 卫龙出海用户画像分析

> 以卫龙品牌为例，分析中国休闲零食在东南亚、北美、欧洲三大海外市场的用户画像差异

## 项目背景

卫龙是中国辣味休闲零食头部品牌，近年来加速海外布局。本报告聚焦核心问题：

- 不同市场的消费者是谁？（华人 vs 本地人）
- 他们为什么买卫龙？（怀旧 vs 尝鲜）
- 购买行为和渠道有何不同？
- 不同市场偏好的产品有何差异？

## 分析框架

```
消费者类型 × 三大市场 × 五大维度

消费者：海外华人 / 本地尝鲜者
市场：东南亚 / 北美 / 欧洲
维度：人口画像 · 购买行为 · 渠道偏好 · 产品偏好 · 消费动机
```

## 技术栈

- **Python** (pandas, matplotlib, seaborn) — 数据构建与可视化
- **行业数据** — 中国零食出海报告、侨民人口统计、电商平台数据
