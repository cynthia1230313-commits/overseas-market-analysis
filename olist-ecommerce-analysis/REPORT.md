# Olist 巴西电商数据分析报告

> **分析师：** 林彤 (TONG LIN)
> **数据来源：** Brazilian E-Commerce Public Dataset by Olist (Kaggle)
> **分析周期：** 2016年9月 — 2018年8月
> **数据规模：** 1,550,922 行原始数据 × 9 张业务表

---

## 1. 执行摘要 (Executive Summary)

本报告基于 Olist 巴西电商平台 **96,478 笔真实订单**（涉及 93,358 名用户、73 个品类、2,970 个卖家）进行全方位数据分析。核心发现：

- **GMV R$1,328 万**，平台处于高速扩张期，月均 GMV 从起步期的 R$1.4 万增长至成熟期 R$85.5 万
- **配送时效是用户满意度的第一驱动力**：5 天内送达的订单评分 4.35，超过 30 天骤降至 2.16（r = -0.304, p < 0.001）
- **价格不直接影响评分**（r = 0.003, p = 0.28），用户更关注服务体验而非价格本身
- **用户复购率仅 3.0%**，复购用户中位数 28 天内完成第二次购买，激活期窗口极短
- **6 个品类贡献 46% 营收**（健康美容、手表礼品、床浴、运动休闲、电脑配件、家具装饰）
- **Top 5% 卖家贡献 53% 营收**，卖家生态高度集中
- **圣保罗一州占 38.4% 营收**，Top 3 州占 63.4%

### 2.2 数据清洗要点

- **缺失值处理：** order_items 3 行缺失已删除，products 维度字段 2% 缺失按品类中位数填充
- **异常值识别：** 价格/运费分布合理，未发现系统性数据录入错误
- **时间范围过滤：** 剔除 97% 已交付订单外的非正常状态（已取消/不可用等）
- **最终分析表：** 合并 8 张表，110,840 行明细数据

## 3. 核心分析发现

### 3.1 营收与增长趋势

| 指标 | 数值 |
|------|------|
| 总 GMV | R$ 13,279,837 |
| 总订单数 | 96,478 |
| 总用户数 | 93,358 |
| 平均客单价 (AOV) | R$ 159.86 |
| 峰值月 | 2017-11 (R$ 995,200) |
| 成熟期月均 GMV | ~R$ 855,000 |

**洞察：** 平台经历显著增长期。起步阶段（2016Q4）月均 GMV 仅 R$1.4 万，成熟期（2018）月均 R$85.5 万。年末（11月）为旺季，与巴西假日购物季吻合。

### 3.2 品类 ABC 分析

| 分类 | 品类数 | 营收占比 | 代表品类 |
|------|--------|---------|---------|
| A 类 (Top 50%) | 6 | 45.9% | health_beauty, watches_gifts, bed_bath_table |
| B 类 (50-80%) | 10 | 33.2% | housewares, cool_stuff, auto, toys |
| C 类 (80-100%) | 57 | 20.9% | 长尾品类，单品贡献 < 1% |

**洞察：** 品类呈现典型的帕累托分布。仅 6 个品类贡献近半营收，长尾品类数量多但效率低。平台应聚焦 A 类品类的供应链优化，同时评估 C 类品类的去留。

### 3.3 配送时效对评分的影响（本次分析的核心发现）

| 配送时效 | 平均评分 | 订单量 | 占比 |
|---------|---------|--------|------|
| 0-5 天 | **4.35** | 22,175 | 20.1% |
| 6-10 天 | **4.26** | 37,602 | 34.1% |
| 11-15 天 | 4.16 | 24,202 | 22.0% |
| 16-20 天 | 4.01 | 12,179 | 11.1% |
| 21-30 天 | 3.52 | 9,378 | 8.5% |
| 30 天以上 | **2.16** | 4,386 | 4.0% |

```
配送时效 vs 评分的 Pearson 相关系数：r = -0.304, p < 0.001 ***
```

**这是本次分析最清晰的因果信号。** 配送每延迟一个档位，评分平均下降 0.35-0.5 分。10 天是一个关键阈值——10 天内送达的订单评分维持在 4.26 以上，超过 10 天后评分加速下滑。

**业务含义：** 与其降价促销，不如投资物流提速。将「10 天送达率」从当前的 54.3% 提升至 80%，预计可拉动整体评分上升 0.2-0.3 分。

### 3.4 价格与评分：几乎无关

| 价格区间 | 平均评分 |
|---------|---------|
| Q1 (最低 20%) | 4.08 |
| Q2 | 4.07 |
| Q3 | 4.05 |
| Q4 | 4.10 |
| Q5 (最高 20%) | 4.10 |

```
价格 vs 评分的 Pearson 相关系数：r = 0.003, p = 0.279 (不显著)
```

**洞察：** 价格高低不决定用户满意度。这与直觉相反——通常认为"花更多钱的人期望更高、更易失望"。在巴西电商语境下，用户对**服务体验（物流、包装、售后）的敏感度远高于对价格本身的敏感度**。

### 3.5 地理集中度

| 指标 | 数值 |
|------|------|
| SP (圣保罗) | 38.4% |
| RJ (里约) | 13.5% |
| MG (米纳斯吉拉斯) | 11.5% |
| **Top 3 合计** | **63.4%** |
| HHI 指数 | 1,897 (中度集中) |

**洞察：** 东南部三州是绝对核心市场，与巴西经济地理高度一致。平台在东北部和北部州的渗透率仍有较大提升空间。

### 3.6 支付行为

| 支付方式 | 订单量 | 占比 | 平均客单价 |
|---------|--------|------|-----------|
| 信用卡 | 73,941 | 76.7% | R$ 182.21 |
| Boleto (银行票据) | 19,191 | 19.9% | R$ 176.23 |
| Voucher | 1,861 | 1.9% | R$ 121.29 |
| 借记卡 | 1,484 | 1.5% | R$ 149.18 |

- **分期付款渗透率：** 52.2%，平均分期 4.9 期
- 信用卡 + 分期是巴西电商的核心支付场景

### 3.7 卖家生态集中度

| 卖家分组 | 营收占比 |
|---------|---------|
| Top 1% | 25.5% |
| Top 5% | 53.0% |
| Top 20% | 82.3% |

**洞察：** 平台呈现头部卖家主导格局。这对平台而言是一把双刃剑——头部卖家带来稳定 GMV，但过度依赖少数卖家的平台抗风险能力较弱。建议平台关注中部卖家的孵化。

### 3.8 用户留存与复购

| 指标 | 数值 |
|------|------|
| 整体复购率 | **3.0%** |
| Month 1 留存 | 5.5% |
| Month 6 留存 | 0.3% |
| 复购中位间隔 | **28 天** |
| 平均复购间隔 | 81 天 |

**洞察：** 这是平台最大的增长瓶颈。3% 的复购率显著偏低（健康电商通常在 20-30%）。但数据也揭示了一个积极的信号：**复购用户的决策窗口极短**——中位数仅 28 天内就完成第二次购买。这意味着如果能在首单后 2-4 周内通过邮件/推送/优惠券进行精准触达，有较大机会提升复购转化。

## 4. 业务建议 (Actionable Recommendations)

### 优先级 P0 — 立刻执行
1. **建立「首单后 14-28 天」复购激活机制**
   - 依据：复购中位间隔 28 天，首月留存 5.5%
   - 动作：首单后第 14 天推送品类关联推荐，第 21 天发送限时优惠券

### 优先级 P1 — 本季度完成
2. **物流履约优化：将 10 天送达率从 54% 提升至 80%**
   - 依据：10 天内评分 4.26+ vs 30 天+ 评分 2.16（差距 2.2 分）
   - 动作：与头部物流商谈判仓储前置，SP/RJ/MG 三州实现 3 日达

3. **A/B/C 品类策略分群运营**
   - A 类（6 个品类）：保障供应链稳定性，设置安全库存
   - B 类（10 个品类）：通过定价/推荐策略提升至 A 类
   - C 类（57 个品类）：评估去留，设置最低绩效门槛

### 优先级 P2 — 中长期
4. **卖家生态优化**：扶持中部卖家（第 20-50 百分位），降低头部依赖
5. **地理扩张**：评估东北部/北部州的物流可行性与获客模型
