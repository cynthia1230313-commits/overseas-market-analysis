# 出海数据分析项目集

> 林彤 (TONG LIN) — Data-driven overseas market analysis projects

## 项目列表

### 1. DTC 品牌用户分群与 EDM 策略优化
**`user-segmentation-edm/`**

基于 RFM 模型对北美 DTC 美妆品牌进行用户精细化分群，输出差异化 EDM 营销策略。Champion 用户（23.8%）贡献 54.2% 收入，针对 6 个用户群体设计差异化邮件策略。

- **技术栈：** Python + SQL + RFM 模型
- **产出：** 数据分析脚本 + SQL 查询 + 可视化图表 + 策略报告

### 2. 出海市场用户画像对比：北美 vs 东南亚
**`market-user-profile/`**

对比分析美妆品类在北美与东南亚的用户画像差异。北美高客单低频（$41×5.6次），东南亚低客单高频（$18×8.2次），渠道与决策因素完全不同。

- **技术栈：** Python + 行业数据整合
- **产出：** 5 张可视化图表 + 分市场策略建议

### 3. 卫龙出海用户画像分析
**`weilong-overseas-profile/`**

以卫龙为例，分析中国零食在东南亚、北美、欧洲的用户画像。发现东南亚本地消费者已超过华人（55% vs 45%），各市场增长瓶颈不同。

- **技术栈：** Python + 行业数据整合
- **产出：** 5 张可视化图表 + 分市场产品/渠道/营销策略
