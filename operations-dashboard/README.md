# 跨境电商运营看板 — Streamlit Dashboard

> 交互式数据看板，面向北美与东南亚市场的 DTC 美妆品牌运营场景

## 功能概览

| Tab | 内容 | 图表 |
|-----|------|------|
| 📈 总览 | KPI 卡片、月度趋势、客单价、订单量 | 折线图、饼图、柱状图 |
| 👥 用户分析 | RFM 分群、用户增长趋势 | 双轴图、面积图 |
| 📦 商品分析 | 销量对比、毛利率矩阵 | 柱状图、气泡图 |
| 📢 营销渠道 | 渠道偏好、转化率、CAC | 横向柱状图、热力图 |

## 技术栈

- **Streamlit** — Web 框架
- **Plotly** — 交互式图表
- **Pandas / NumPy** — 数据处理

## 使用方法

```bash
# 安装依赖
pip install streamlit plotly pandas numpy

# 启动看板
streamlit run app.py

# 浏览器打开 http://localhost:8501
```

## 特色

- 全局筛选器：可按市场、日期范围动态筛选
- 全交互式图表：悬停查看数据、缩放、导出
- 模拟数据基于真实市场基准（北美 AOV ~$42 / 东南亚 ~$18）
- 适合面试现场演示

## 在线部署
